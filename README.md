# CSES Problem Set Solutions

My solutions to the [CSES Problem Set](https://cses.fi/problemset/).

## Overview

- **Total solutions:** 193
- **Language:** C++
- **Judge:** [cses.fi](https://cses.fi/)

## File naming

Each file is named by its CSES problem ID (e.g. `1068.cpp` corresponds to problem [1068](https://cses.fi/problemset/task/1068) on CSES).

## Coding style

Solutions are written in competitive programming style:

- `#include <bits/stdc++.h>` for a single-header include
- Common macros for brevity (`ll`, `fr`, `pb`, `F`/`S`, etc.)
- Fast I/O with `ios_base::sync_with_stdio(false)` and `cin.tie(NULL)`
- Focus on correctness and speed rather than production readability
